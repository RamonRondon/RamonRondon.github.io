Iniciar Sesión Administrador

Para publicar artículos en tu blog de forma directa y segura en GitHub Pages, ingresa tu Token de Acceso Personal (PAT) de GitHub.

Tu token se guarda de manera segura y local en tu navegador (localStorage).

¿Cómo obtenerlo? 1. Ve a GitHub -> Settings -> Developer Settings -> Personal Access Tokens -> Fine-grained tokens. 2. Genera un nuevo token dando acceso de Read & Write a Contents para tu repositorio RamonRondon.github.io.

📷 Insertar Imágenes

Puedes pegar imágenes directamente desde tu portapapeles o arrastrarlas al editor:

  • Copiar y Pegar: Copia cualquier imagen en tu portapapeles y presiona Ctrl + V dentro del editor de contenido.
  • Arrastrar y Soltar: Arrastra un archivo de imagen desde tu computadora y suéltalo dentro del editor.

Nota: Las imágenes se guardan de forma temporal en tu navegador y se suben permanentemente a tu repositorio de GitHub solo cuando haces clic en "Guardar y Publicar".

📐 Fórmulas Matemáticas (LaTeX)

El blog soporta fórmulas matemáticas con la notación de LaTeX usando la librería KaTeX:

  • Fórmula en línea (Inline): Rodea la fórmula con un solo símbolo de dólar $.
    Ejemplo: $a^2 + b^2 = c^2$. Se verá integrada en el texto.
  • Fórmula en bloque (Destacada): Rodea la fórmula con dos símbolos de dólar $$.
    Ejemplo: $$ c = \sqrt{a^2 + b^2} $$ Se mostrará en su propio bloque con el diseño de **hoja cuadriculada rasgada de cuaderno**.
